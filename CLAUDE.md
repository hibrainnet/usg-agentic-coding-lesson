# 프로젝트

이 프로젝트는 컴퓨터공학 전공 4학년 대상으로 "Claude Code를 활용한 Agentic Coding"이라는 주제로 6시간 온라인 강의 자료 구축이 목적이다. 강의는 시간마다 학생들이 실습을 따라할 수 있는 자료를 준비해서 실습과 함께 진행하는 형태로 진행한다.

# 프로젝트 구조
- LESSONS.md 파일은 프로젝트 주제에 해당하는 수업 내용을 요약한 파일이다.
- lessons 디렉토리는 6시간 주제별로 해당 수업 내용이 들어가는 디렉토리이다.
- 각 lesson 디렉토리 밑에는 docs, tutorials, examples, assets 디렉토리가 존재하고 각각 필요한 자료가 저장되는 구조이다.
- instructor 디렉토리는 강사 전용 자료(강의 노트, 교안, 모범 답안)가 저장된다. `.gitignore`에 등록되어 있으며 학생 배포 저장소에 포함되지 않는다.

# 자료 작성 규칙
- 학생에게 공개할 자료는 `lessons/` 디렉토리에 작성한다.
- 강사만 볼 자료(강의 노트, 교안, 실습 모범 답안, 운영 가이드)는 반드시 `instructor/` 디렉토리에 작성한다.
- `instructor/` 디렉토리의 내용은 절대 `lessons/` 에 복사하거나 학생에게 직접 공유하지 않는다.
