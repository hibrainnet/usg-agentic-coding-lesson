# 5교시. 자동화와 확장: 커스텀 커맨드·훅·MCP·서브에이전트

> 4교시까지 에이전트와 함께 개발하는 방법을 익혔다면, 이번 시간에는 **반복 작업을 자동화**하고
> 에이전트의 능력을 **외부 도구까지 확장**합니다.
> 한 번 설정해두면 매번 타이핑하지 않아도 되는 나만의 워크플로우를 만들어봅니다.

## ⏱️ 이번 시간(60분) 구성

| 시간 | 내용 | 형태 | 자료 |
|------|------|------|------|
| 0~5분 | 오리엔테이션 + 4교시 복습 | 강의 | — |
| 5~20분 | **커스텀 슬래시 커맨드** — 자주 쓰는 작업을 명령어로 | 강의 + 따라하기 | `docs/01-custom-commands.md` · `tutorials/01-custom-slash-command.md` |
| 20~35분 | **훅(Hooks)** — 특정 이벤트에 자동 실행 | 강의 + 따라하기 | `docs/02-hooks-mcp-subagents.md` · `tutorials/02-hooks-setup.md` |
| 35~55분 | **MCP 서버 연결 & 서브에이전트** — 외부 도구 통합 | 강의 + 따라하기 | `tutorials/03-mcp-subagent.md` |
| 55~60분 | Q&A + 다음 시간 예고 | — | — |

## 🎯 학습 목표

- 자주 반복하는 작업을 커스텀 슬래시 커맨드로 저장하고 재사용할 수 있다.
- 훅을 설정해서 특정 시점(작업 완료, 파일 저장 등)에 자동 동작을 추가할 수 있다.
- MCP 서버의 개념을 이해하고 공개 MCP 서버 1개를 연결해 활용할 수 있다.
- 서브에이전트로 탐색 작업을 위임하는 패턴을 경험할 수 있다.
