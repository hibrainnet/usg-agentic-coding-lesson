# hello-world (1교시 첫 실습 예제)

이 폴더는 1교시 첫 실행 실습에서 사용하는 **빈 연습 공간**입니다.

## 사용법

이 폴더에서 Claude Code를 실행해 첫 대화를 연습합니다.

```bash
cd lessons/01-intro-setup/examples/hello-world
claude
```

그런 다음 튜토리얼(`tutorials/01-install-and-first-run.md`)의 STEP 5부터 따라 하세요.

> 💡 학생들은 보통 홈 디렉토리에 `hello-claude` 폴더를 새로 만들어 연습합니다.
> 이 폴더는 강의 자료 안에서 동일하게 연습하고 싶을 때 사용하는 대안입니다.
