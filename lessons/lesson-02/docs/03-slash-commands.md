# 슬래시 커맨드 & CLI 개요

Claude Code 대화 중 `/`로 시작하는 커맨드로 세션을 제어합니다.
입력창에서 `/`를 입력하면 자동완성 목록이 표시됩니다.

> 전체 커맨드 상세 설명·자습 실습 → `04-slash-commands-deepdive.md`

---

## 수업 중 자주 쓰는 커맨드

| 커맨드 | 동작 | 언제 |
|--------|------|------|
| `/help` | 사용 가능한 커맨드 목록 보기 | 무엇이 있는지 확인할 때 |
| `/clear` | 대화 맥락 초기화 (CLAUDE.md 유지) | **새 작업 시작 시** |
| `/compact` | 긴 대화를 요약·압축 (맥락 유지) | 같은 작업 중 대화가 길어졌을 때 |
| `/init` | 현재 프로젝트 분석 후 `CLAUDE.md` 초안 생성 | 프로젝트 처음 설정할 때 |
| `/usage` | 현재 세션 토큰 사용량·비용 확인 | 비용 확인 시 (`/cost`는 별칭) |
| `/status` | 모델·계정·연결 상태 확인 | 인증 문제 의심 시 |
| `/model` | 사용 모델 확인·변경 | 속도·품질 조절 시 |
| `/plan` | 계획 모드 진입 | 복잡한 작업 사전 설계 시 |
| `/feedback` | 버그 신고·피드백 제출 | Claude Code 자체 문제 시 |
| `/exit` | 세션 종료 | 작업 완료 후 |

## CLI 플래그 — `claude` 실행 방법

```bash
claude                      # 빈 대화 세션 시작
claude "프롬프트"            # 첫 메시지 포함해서 시작
claude --continue  (-c)     # 가장 최근 대화 이어서
claude --resume    (-r)     # 대화 목록 선택해서 재개
claude --print "..."  (-p)  # 비대화형: 응답만 출력하고 종료
claude --model <모델명>     # 특정 모델로 시작
claude --add-dir <경로>     # 작업 디렉토리 추가
```

## 권한(Permission) 모드

파일 수정·명령 실행 전 선택지:

| 선택 | 동작 |
|------|------|
| 이번만 허용 | 한 번만 승인 |
| 이 파일 앞으로 허용 | 같은 파일 자동 승인 |
| 이 명령 앞으로 허용 | 같은 명령 자동 승인 |
| 거부 | 에이전트가 다른 방법 탐색 |

**ESC** — 작업 즉시 중단
